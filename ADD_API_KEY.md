# Добавление вашего 2GIS API ключа

## Шаг 1: Создайте файл .env

В папке `frontend` создайте файл `.env`:

```bash
cd frontend
echo VITE_2GIS_API_KEY=ваш_реальный_api_ключ > .env
```

Или создайте файл вручную со следующим содержимым:

```
VITE_2GIS_API_KEY=ваш_реальный_api_ключ
```

## Шаг 2: Замените ключ

Замените `ваш_реальный_api_ключ` на API ключ, который вы получили на dev.2gis.com

Пример:
```
VITE_2GIS_API_KEY=rudjaq6542
```

## Шаг 3: Перезапустите проект

```powershell
docker-compose down
docker-compose up --build
```

## Проверка

1. Откройте админ панель: http://localhost:3000/admin
2. Войдите с учетными данными администратора
3. Перейдите в "Зоны доставки"
4. Карта должна загрузиться!

## Если карта не загружается

Проверьте:
- ✅ API ключ скопирован правильно (без лишних пробелов)
- ✅ Файл `.env` находится в папке `frontend`
- ✅ Проект перезапущен после создания `.env`
- ✅ API ключ активен на dev.2gis.com

## Где найти ваш API ключ

1. Перейдите на https://dev.2gis.com/
2. Войдите в аккаунт
3. Откройте "Мои проекты"
4. Выберите ваш проект
5. Скопируйте API ключ из раздела "Ключи"

---

**Примечание**: Файл `.env` автоматически игнорируется git, поэтому ваш ключ не попадет в репозиторий.
