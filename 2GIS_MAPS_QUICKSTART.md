# Быстрый старт с 2GIS Карты

## Шаг 1: Получите API ключ

1. Перейдите на https://dev.2gis.com/
2. Зарегистрируйтесь или войдите в аккаунт
3. Перейдите в раздел "Мои проекты"
4. Создайте новый проект и выберите "Maps API 2.0"
5. Скопируйте полученный ключ

## Шаг 2: Настройте проект

Создайте файл `frontend/.env`:

```bash
VITE_2GIS_API_KEY=ваш_api_ключ_здесь
```

## Шаг 3: Используйте зоны доставки

1. Запустите проект: `docker-compose up`
2. Войдите в админ панель: http://localhost:3000/admin
3. Перейдите в "Зоны доставки"
4. Карта загрузится автоматически!

## Возможности

✅ Визуализация всех зон на карте с цветами  
✅ Рисование новых зон прямо на карте  
✅ Редактирование существующих зон  
✅ Интерактивные подсказки при клике на зону  
✅ Автоматическое масштабирование карты  

## Без API ключа

Если не добавите API ключ, страница будет работать, но вместо карты увидите сообщение об ошибке загрузки. Все остальные функции (список зон, редактирование) работают нормально.
